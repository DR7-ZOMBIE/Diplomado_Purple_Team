## 🎯 ¿De qué trata este diplomado?

Este diplomado está diseñado para formar profesionales en ciberseguridad con enfoque **Purple Team**, combinando habilidades de **hacking ético (Red Team)** y **defensa activa y forense (Blue Team)**.

A lo largo de ocho módulos, los participantes desarrollan capacidades técnicas para simular ataques, detectar vulnerabilidades, proteger infraestructuras críticas y realizar análisis forense digital con herramientas de clase mundial.

---

## 📚 Módulos del programa

| Módulo                     | Contenido principal                                                |
|---------------------------|--------------------------------------------------------------------|
| **1. OSINT**              | Google Dorks, Shodan, perfiles digitales, enumeración activa/pasiva |
| **2. Web Pentesting**     | XSS, SQLi, OWASP Top 10, Burp Suite, inclusión de archivos         |
| **3. Redes**              | Sniffing, ARP Spoofing, Evil Twin, análisis de tráfico             |
| **4. Seguridad en la nube** | Gestión IAM, DLP, protección en herramientas colaborativas         |
| **5. Móvil**              | APK reversing, Frida, bypass de root y SSL pinning                 |
| **6. Explotación**        | Buffer overflows, ROP, escalación de privilegios                   |
| **7. Directorio Activo**  | BloodHound, Kerberoasting, GPOs inseguras                          |
| **8. Forense Digital**    | Cadena de custodia, malware, memoria RAM, estándares NIST/ISO      |
